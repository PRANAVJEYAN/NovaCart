# E-Commerce Platform

Modern e-commerce platform with React + Vite frontend and Express + MongoDB backend.

## Setup

1. Install all dependencies:
   npm run install:all
2. Or install separately:
   npm run install:client
   npm run install:server
3. Copy environment files:
   cp client/.env.example client/.env
   cp server/.env.example server/.env
4. Start backend:
   npm run dev:server
5. Start frontend:
   npm run dev:client

## Deployment

- Frontend: Vercel/Netlify compatible Vite build.
- Backend: Render/Railway Node.js service.
- Database: MongoDB Atlas.
